# SOVEREIGN-CORE-PSC (Project Sovereign Congo)

## 📡 Mission
Système mobile de renseignement d'origine électromagnétique (**SIGINT**) et de guerre électronique (**ELINT**) optimisé pour la détection et l'analyse des flux de données financiers (Mobile Money).

## 🛠️ Spécifications Techniques
- **Cœur :** Kotlin / JVM (Stabilité industrielle)
- **Pilote RF :** LibUHD 4.9.0 (Souveraineté sur le matériel USRP B210)
- **Analyse :** Python 3.12 (Spectrogrammes et rapports PDF)
- **Plateforme :** Android / Termux (Mobilité tactique)

## 🚀 Utilisation
1. Brancher l'USRP B210 via OTG.
2. Autoriser l'accès : `termux-usb -r /dev/bus/usb/...`
3. Lancer la mission : `./mission_monolithe.sh`
