# 🛡️ MANUAL DE OPERACIONES - SOVEREIGN CORE v4.2
## SISTEMA INTEGRADO DE GUERRA ELECTRÓNICA Y MESH - FARDC

### 1. ARQUITECTURA DE RED DISTRIBUIDA (NEW: MESH RELAY)
La v4.2 permite la interconexión de múltiples unidades para cubrir zonas de sombra.
* **Modo Relé**: Cada smartphone/SDR actúa como repetidor.
* **Alcance Extendido**: La red puede cubrir provincias enteras mediante saltos sucesivos.

### 2. MÓDULOS DE INTELIGENCIA ACTIVA
* **📍 GEO-FUSION**: Localización precisa mediante triangulación TDOA.
* **🔍 COMINT DECODER**: Intercepción de metadatos de drones y comunicaciones enemigas.
* **⚠️ IA THREAT ANALYZER**: Clasificación automática del nivel de peligro (0-100%).

### 3. PROCEDIMIENTO DE DESPLIEGO EN RED
1. Iniciar unidades en modo "MESH_ACTIVE".
2. Validar el enlace en el HUD (Icono de malla azul).
3. El sistema fusionará automáticamente las amenazas de todas las unidades en un solo mapa táctico.

### 4. MODOS DE COMBAT ACTUALIZADOS
* **SILENT WATCH**: Vigilancia pasiva y relé invisible.
* **AUTO DEFENSE**: Respuesta automática coordinada en toda la red Mesh.
* **ELECTRONIC ASSAULT**: Ataque electrónico masivo sincronizado (Jamming de Red).
