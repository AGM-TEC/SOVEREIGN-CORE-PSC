# 📑 SOP-GEO-01 : TRIANGULATION ET FUSION SIGINT
## VERSION TACTIQUE 4.1 - FARDC

### 1. CONFIGURATION DU RÉSEAU
* Déployer au minimum 3 unités distantes de 800m à 1500m.
* Activer le "MeshSyncEngine" sur chaque terminal.
* Vérifier que le GPS de chaque appareil est actif et stable.

### 2. PROTOCOLE D'INTERCEPTION
1. Identifier la fréquence suspecte sur le HUD.
2. L'unité maître initie la "Requête de Fusion".
3. Les unités esclaves renvoient les horodatages (timestamps) du signal capté.

### 3. ACTION TACTIQUE
* Si la précision est < 20m : Transmission immédiate des coordonnées à l'unité d'artillerie ou au drone d'interception.
* Si le signal est mobile : Suivi continu en mode "Assault Tracking".
