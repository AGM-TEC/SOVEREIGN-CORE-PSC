# NOTES DE RELEASE - SOVEREIGN v7.1-GOLD

## DESCRIPTION
Version finale de l'arsenal cyber-offensif pour l'unité SIGINT. Ce système assure la capture financière, la neutralisation de drones et l'exfiltration de données hors-ligne.

## CHANGELOG FINAL
- **DE-TRAPPING :** Suppression complète des mécanismes d'auto-destruction et des "canaris" des versions 4 et 5.
- **MESH-KINETIC :** Activation du maillage P2P décentralisé pour 15 agents (Communication sans réseau).
- **SKYGUARD :** Armement du module de guerre électronique anti-drone (Standard 2.4/5.8GHz).
- **SIGNAL-GHOST :** Déploiement du module d'exfiltration Air-Gap par ondes électromagnétiques.
- **BLINDAGE :** Intégration du moteur de mutation polymorphe pour évasion EDR/AV.

## INSTRUCTIONS DE DÉPLOIEMENT
Utiliser la clé de chiffrement AES-256 fournie pour extraire l'archive dans le dossier cible.
